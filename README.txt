Данная программа создана для описания графических примитивов и выполнения различных манипуляций с ними, используя System.Drawing.Common
Таких как:
- создание примитивов (прямоугольник, круг, равносторонний треугольник);
- проверка, лежит ли заданная точка внутри примитива;
- перемещение;
- изменение размеров;
- связывание примитивов линиями.

В нижней части формы расположены кнопки с различными действиями.

Для создания примитива сначала выбираем, какой примитив хотим создать (прямоугольник, круг, равносторонний треугольник).
После этого нажимаем в любое место на форме, и примитив создаётся.

Чтобы проверить, лежит ли заданная точка внутри примитива, выбираем нужное действие на кнопках и также нажимаем в любое место на форме.
После этого появится сообщение, если точка находится внутри примитива.

При выборе действия "Перемещение" нажимаем на примитив, который хотим переместить, а затем указываем новое место на форме.

Изменение размера работает аналогично перемещению.
Выбираем примитив, а затем, используя ось Y, указываем точку, на какое расстояние от центра хотим изменить размер примитива (для круга и треугольника).
Для прямоугольника сначала выбираем его, затем по оси Y:
- вверх от центра — увеличиваем размер,
- вниз от центра — уменьшаем.

Для связывания примитивов линиями нужно сначала нажатием мыши выбрать первый примитив, а затем второй.

Также имеется кнопка "Прекращение действия", которая позволяет временно отключить взаимодействие с формой, пока не будет выбрано другое действие.
